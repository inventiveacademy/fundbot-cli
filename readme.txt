INSTRUCTIONS ON USE!

To Start in Command Line:
"node index.js"

To Get All:
"node index.js -g"

To Get by ID:
"node index.js -g -i idnumber"

To "Soft" delete:
"node index.js -d idnumber"

To Query Search:
"node index.js -q 'category':'searchparameter'"

To Post New Applicant:
"node index.js -p"
Then:
Follow Prompts

To Edit Applicant:
"node index.js -u idnumber"
